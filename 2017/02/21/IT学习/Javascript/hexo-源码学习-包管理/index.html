hexo 使用的第三方模块一览 (package.json文件分析)

开发环境依赖 (devDependencies)

模块名 主页 作用
chai https://github.com/chaijs/chai 断言框架，用于测试
chai-as-promised https://github.com/domenic/chai-as-promised chai 模块的 promise/a+ 扩展。(简化js异步回调为链式写法)
eslint https://github.com/eslint/eslint javascript 代码检查工具
eslint-config-hexo https://github.com/hexojs/eslint-config-hexo hexo 工程的个性化配置
hexo-renderer-marked https://github.com/hexojs/hexo-renderer-marked hexo 的 markdown 文件渲染引擎
istanbul https://github.com/gotwarlost/istanbul 代码覆盖率检测工具
jscs https://github.com/jscs-dev/node-jscs js 代码格式化工具(该模块已合并到 eslint 模块中)
jscs-preset-hexo https://github.com/hexojs/jscs-preset-hexo hexo 工程的 jscs 个性化配置
mocha https://github.com/mochajs/mocha nodejs 单元测试框架
rewrite https://github.com/teknopaul/rewrite url 重定向模块
sinon https://github.com/sinonjs/sinon 辅助测试工具，可以模拟数据库，冒烟等场景

正式环境依赖 (dependencies)

模块名 主页 说明
abbrev https://github.com/isaacs/abbrev-js ruby 上 abbrev 的 nodejs 实现，类似自动补全的功能，多用于命令行的命令补全(tab健功能)
archy https://github.com/substack/node-archy 把 json 对象用漂亮的树状结构打印出来
bluebird https://github.com/petkaantonov/bluebird 是一种 promise 的实现，简化 nodejs 事件回调、捕捉、书写的辅助模块
chalk https://github.com/chalk/chalk 使控制台的输出变得五颜六色
cheerio https://github.com/cheeriojs/cheerio server 版的 jquery
deep-assign https://github.com/sindresorhus/deep-assign 递归赋值的工具模块
hexo-cli https://github.com/hexojs/hexo-cli hexo 的命令行封装
hexo-front-matter https://github.com/hexojs/hexo-front-matter 使页面支持在页头定义参数
hexo-fs https://github.com/hexojs/hexo-fs hexo 文件系统处理模块
hexo-i18n https://github.com/hexojs/hexo-i18n 国际化模块
hexo-log https://github.com/hexojs/hexo-log 日志模块
hexo-util https://github.com/hexojs/hexo-util hexo 扩展的常用工具集合
js-yaml https://github.com/nodeca/js-yaml YAML 处理工具模块
lodash https://github.com/lodash/lodash js 常用工具结合
minimatch https://github.com/isaacs/minimatch 文件名匹配工具 如: a.js, b.js 匹配为 *.js
moment https://github.com/moment/moment 日期转换模块
moment-timezone https://github.com/moment/moment-timezone 使 moment 支持时区
nunjucks https://github.com/mozilla/nunjucks 模板引擎
pretty-hrtime https://github.com/robrich/pretty-hrtime 格式化时间
strip-indent https://github.com/sindresorhus/strip-indent 处理字符串开头的空字符
swig https://github.com/paularmstrong/swig 模板引擎
swig-extras https://github.com/paularmstrong/swig-extras swig 的一些标签扩展
text-table https://github.com/substack/text-table 格式化字符串表格
tildify https://github.com/sindresorhus/tildify 将全路径格式化为～
titlecase https://github.com/rvagg/titlecase 将段落首字母大写
warehouse https://github.com/tommy351/warehouse 基于 JSON 的数据库

执行任务 (scripts)

任务 指令 说明
eslint eslint . 代码检查
jscs jscs . 代码格式化
test mocha test/index.js 自动化测试
test-cov istanbul cover –print both _mocha – test/index.js 覆盖率检查
